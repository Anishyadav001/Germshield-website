# GermShield - Premium Cleaning & Hygiene Solutions

GermShield is a highly professional, responsive, and modern B2B catalog platform designed for commercial cleaning and hygiene solutions. This platform showcases premium commercial products engineered for high-footfall environments like corporate hubs, healthcare systems, malls, and multi-shift complexes.

🔗 **Live Link:** [https://germshield.netlify.app/](https://germshield.netlify.app/)

---

## 🚀 Key Features

- **Premium B2B UI/UX:** A modern, clean, and highly professional layout tailored for corporate clients.
- **Dynamic Catalog Matrix:** Comprehensive product categories including:
  - Advanced Liquid Cleaning Solutions
  - Premium Phenyl Series
  - Advanced Institutional Specialties (Kitchen Degreasers, Surface Disinfectants)
  - Housekeeping Tools & Auxiliary Supplies
- **Instant Procurement Integration:** Direct WhatsApp-linked call-to-actions (`Request Details`) for streamlined institutional ordering.
- **Corporate Consultation Form:** An integrated interactive modal for lead generation and monthly volume estimations.
- **Fully Responsive Design:** Seamless browsing experience across mobile, tablets, and desktops.

---

## 🛠️ Tech Stack Used

- **HTML5:** For structured layout.
- **CSS3:** Custom styling with modern UI variables and responsive media queries.
- **JavaScript (Vanilla):** For interactive modal controls, smooth scrolling, and dynamic features.
- **Netlify:** For lighting-fast hosting and continuous deployment.
